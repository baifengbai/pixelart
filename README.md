# pixelart

R Package to make Pixel Art models. [[Blog post introducing the package]](https://privefl.github.io/blog/shiny-app-for-making-pixel-art-models/)

```r
# Installation
devtools::install_github("privefl/pixelart")

# Run Shiny App
pixelart::run_app()
```
